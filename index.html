Getting Started

System Requirements

  • M7/Movian Media Center: Version 7.0.157 or higher
  • PlayStation 3: CFW with Movian installed
  • Android: Movian app from movian.eu
  • Internet Connection: Minimum 10 Mbps for SD, 20 Mbps for HD

Installation Guide

Important: Note: Open the side menu from any page inside Streamian and select "Update Streamian" periodically to recieve updates.

Method 1: Official Repository (Recommended)

  1. Launch Movian/M7 Media Center
  2. Navigate to Plugins â†’ Browse Available Plugins
  3. Select Video Streaming category
  4. Find and select Streamian
  5. Click Install and wait for completion

Method 2: Manual Installation

  1. Download the latest streamian_stable.zip from the Releases tab
  2. Transfer the ZIP file to your device
  3. In Movian/M7, go to Your Device â†’ find Streamian
  4. Click Install and wait for completion

Configuration & Settings

Streaming Settings

Recommended Settings for PlayStation 3

  • Video Quality: HD 1080p
  • H.265 Filter: ALWAYS ENABLED (PlayStation 3 Cannot Display h.265)
  • Minimum Seeders: At leaast 20+ for reliable streaming

Advanced Usage

Creating Custom Addons

Streamian supports custom scraper addons. Create a JavaScript file with the following structure:

JavaScript 
// Example custom addon structure
(function() {
  var service = require('movian/service');
  var http = require('movian/http');
  
  // Register your addon
  service.create('My Custom Addon', 'custom:start', 'video', true, 'icon.png');
  
  // Add scraping logic
  function scrapeContent(query, type) {
    // Your scraping logic here
    return [
      {
        title: "Example Video",
        url: "magnet:?xt=...",
        seeders: 50
      }
    ];
  }
  
  // Export functions
  exports.scrape = scrapeContent;
})();

Library Management

  • Adding Content: Right-click (or long-press) any show/movie and select "Add to Library"
  • Organizing: Library automatically categorizes by type (Shows/Movies/Channels)
  • History: Watch history is maintained separately for on-demand and live channels
  • Backup: Export your library via Settings â†’ Data Management

Troubleshooting

Common Issues:

  • No streams found: Check your addon URLs in settings or ISP blocking
  • Playback errors on PS3: Enable H.265 filter and reduce quality settings
  • Channels not loading: Verify region detection in log (Enable Debug)
  • Movian/M7 Crashing Reinstall Movian/M7

FAQ

General Questions

Q: Is Streamian legal to use?
A: Streamian is a media aggregation plugin. The legality depends on your location and the content you access. Always respect copyright laws.

Q: Does it work on jailbroken PlayStation 3?
A: Yes, Streamian is fully compatible with PS3 running custom firmware with Movian installed.

Q: Can I add my own streaming sources?
A: Yes, via the custom addon slots in settings or by creating M3U playlists for live channels.

Q: How often should I update?
A: Check for updates monthly or when features stop working. Use the "Update Streamian" option in the side menu.

Q: Is there a Discord or support community?
A: Yes, join the discussion at r/movian on Reddit for support and updates.
